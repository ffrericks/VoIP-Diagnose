    Find the IPv4 address and enter it below:

    Subnet check

    Connection speed automatic

    More accurate measurement optional

    1. Press Windows key + R, type cmd and press Enter
    2. Copy and paste the following command and press Enter:
    ping ha.voys.nl -n 20

    Wait until the command completes. Then select all text in the window — click at the very top, scroll down, hold Shift and click at the bottom. Then press Ctrl+C and paste the result below.

    1. Open Terminal (Spotlight: Cmd + Space, type Terminal)
    2. Copy and paste the following command and press Enter:
    ping -c 20 ha.voys.nl

    Wait until the command completes. Then select all text in the window (Ctrl+A), copy it (Ctrl+C) and paste it below.

    Double-NAT detection

    Automatic (CGNAT)
    Traceroute (more accurate)
    1. Press Windows key + R, type cmd and press Enter
    2. Copy and paste the following command and press Enter:
    tracert ha.voys.nl

    Wait until complete (may take 30 seconds). Copy all output and paste below.

    1. Open Terminal (Spotlight: Cmd + Space, type Terminal)
    2. Copy and paste the following command and press Enter:
    traceroute ha.voys.nl

    Wait until complete. Copy all output and paste below.
